Topics for Exam III  - December 11, 2003

Over Chapters 16.1-3, 16.5, 22-24.3, 25.2, 26.1,26.2 and handout

 

 

·       Greedy Method

o     Understand the “greedy choice” principle and how this can be formulated as an invariant

o     Show that the greedy method yields sub-optimal solutions to certain problems

o     Know how to show a problem yields a greedy solution.

o     Understand the Huffman encoding scheme, fractional knapsack, coin changing (16-1), task scheduling with deadlines and all times the same, and container loading problems.

o     To review, do problems 16.2-3,16.2-4,16.2-5,16.3-1 – 16.3-3, 16.3-7, Problem 16-2 a.

 

·       Graphs

o     Know basic definitions

o     Know adjacency list and matrix benefits and drawbacks and implementations.

o     Understand BFS and DFS

 

·       MST

o     Understand and be able to demonstrate Kruskal’s and Prim’s algorithms and their correctness.

o     Know how to perform topological sort and its applications.

o       Review Problems 23.2-1,23.2-2

 

·       Shortest Path

o     Understand and be able to demonstrate Dijkstra’s and Bellman-Ford algorithms and their correctness.

o     Understand how to formulate all pairs shortest path as a dynamic program and as repeated application of single source shortest path.

o     Review  Exercisess 24.1-1,24.3-1,24.3-2    Problem 24-3a , Exercises 25.2-1,25.2-4

o      

·       Maximum flow

o     Understand the maximum flow and min-cut theorem.

o     Know how to construct augmenting flow paths (with forward and backward labelings).

o     Know how to perform vertex splitting for computing the number of vertex-disjoint paths.

o     To review, do problems 26.2-1 through 26.2-4 and 26.2-9.

 

·       NP Completeness and Undecidability

o     Understand P, NP, NPC, and NP-Hard and why they are important.

 

 

·       Pre/Post and Invariants and Testing

o     As always be able to generate invariants that describe progress so far in the program and be able to set up test cases for algorithms.

1)      A topological sort can report the existence of a cycle if one is encountered.

a.      (15 pts) Give topological sort on a directed graph that will label the vertices appearing in the cycles (if any).

 

b.     (5 pts) Show an example of your algorithm’s execution.

 

2)     A maximal cost spanning tree of a weighted graph is a spanning tree with the highest possible cost). 

a.      (15 pts) Construct an algorithm to produce the maximal cost spanning tree.

 

b.     (10 pts) Formulate an invariant that will yield the postcondition of (a).

 

c.      (10 pts) What is the running time complexity of your algorithm for the graph represented by (a) an adjacency list and (b) an adjacency matrix?

 

d.      (5 pts) Show an example of your algorithm’s execution.

 

  

3)    Cycles in a digraph.

a.      (15 pts) Write code to determine if a directed graph has a cycle.  Use either DFS or BFS.

 

b.     (5 pts) Show an example of your algorithm’s execution on a simple digraph.

 

c.      (10 pts) Formulate an invariant that describes what portion of the graph has been explored so far at each step of the execution.

 

4)     The traveling salesman problem is a known NP-Complete problem.  You are given an n vertex weighted graph and the task is to find a cycle that includes all n vertices and has  weight < k for some k.  However, we may approximate a solution to it with a greedy approach. 

a.      (15 pts) Construct a greedy algorithm to produce the minimum cost cycle for the traveling salesperson problem.

 

b.     (10 pts) What is the running time complexity of your algorithm for the graph represented by (a) an adjacency list and (b) an adjacency matrix?

 

c.      (15 pts) Show an example of your algorithm’s execution that shows it does not find the optimal solution.

 

5)    Let G be a directed acyclic graph with n vertices.  Assume the vertices have been labeled 1 through n such that if (i,j) is an arc of G, then i<j.  Let L(i,j) be the length of the edge (i,j).

 

a.      Formulate a dynamic programming recurrence to obtain the longest path in G (15 pts).

 

b.     Give a recurrence for the running time bound on (a) and solve the recurrence (15 pts).
