Course Objectives:  

 

Winter Semester, 2003

 

At the end of the course you should be able to:

 

Test I



·     Design, simulate, and synthesize a digital logic circuit using Mentor graphics schematic capture or VHDL and Leonardo Spectrum, and/or Actel Designer. 

·     Explain the most important trends in IC design that challenge the IC engineer.  For example, explain the importance of Moore’s law, chip complexity, transistor counts, die size, frequency, and power.

·     Identify the most significant contributors to IC cost.  Given design constraints such as time-to-market, speed, size, cost, and market, determine which chip technology (custom, CBIC, MGA, FPGA) is best for your product.  Explain the importance of time-to-market and chip size on the overall cost or profit of an IC design.

·     Explain the function of each element of an ACT1 FPGA (logic module, I/O module, clock buffer, horizontal and vertical interconnect).  Explain how an anti-fuse works.  Explain why a special buffer is needed for the clock.

·     Given a connections to ACT1 logic modules, describe the logic function that is implemented.  Given a logic function, show the connections necessary to perform that function using ACT1 logic modules.  Where possible, use the minimum number of logic modules to implement the logic function.  Show why 2 ACT1 logic modules are needed to implement a single flip-flop.  Given a logic diagram, estimate (without building) the number of ACT1 logic modules required to implement the logic.

·     Given a complex RC circuit, calculate the approximate time delay constant from one point in the circuit to another using Elmore’s constant.  Given a representation of an ACT1 circuit, determine an equivalent RC circuit.  Identify values for circuit parameters within the circuit from tables for the ACT1 part.  Adjust timing calculations based on derating factors like temperature, voltage, and fan-out.

·     Given a rough diagram of connections for a logic circuit within an ACT1 FPGA, calculate the time-delay through an ACTEL logic circuit using timing parameters like tPD, tSUD, tCO, and interconnect delay.  Approximate interconnect delay if exact connections are not given.

·     Explain why a gated clock is a problem.

·     Read and understand basic VHDL code.  Given a VHDL description, draw a block diagram of the resulting hardware or vice-versa.  Recognize VHDL code that will and will not synthesize correctly.  Recognize VHDL code that will synthesize badly (for example, will use too much logic or will instantiate undesired latches). Write synthesizeable VHDL code for simple systems (for example, an ALU).  Roughly estimate the size of synthesized VHDL code.

·     Describe how an NFET or PFET works, including the basic equations for operation, the regions of operation (saturation, etc), how a depletion region is created and why it is so important. Explain the need for a connection to the well.

·     Explain why an NFET is good as a pull-down tranisistor and a PFET is good as a pull-up.  Show voltage levels (gate, source, drain, output) for an NFET or PFET as it tries to pull up or down an external output.  Given plots of voltage levels for a simple logic circuit, predict if the circuit is being pulled-up or pulled-down by an NFET or PFET.  Calculate gate capacitance.

·     Identify the major causes of power dissipation in a digital circuit.  Calculate overall power dissipation given characteristics like gate size, voltage levels, clock speed, rise and fall times.

·     Given a CMOS circuit diagram, describe the logic function performed by the CMOS circuit.  Given a logic equation, draw a CMOS circuit that implements that equation.

·     Explain the steps taken to fabricate an IC.  Show how a sequence of steps applying photo resist, UV light, etchant, etc, can be used to develop a particular layer (for example, the polysilicon layer). Given a chip-layout, draw a cross-section of a the chip at any given point.  For example, the cross-section of a transistor showing the well, diffusion, gate oxide, polysilicon, metal layers, and overglass.  Explain how many processes are implemented to make transistor gates “self-aligned” and why this feature is important.

·     Given a stick-diagram, draw the associated circuit diagram.  Given a circuit diagram, draw the stick diagram for that circuit.  Perform the same operations for a full-layout, including specification of dimensions (for example, specifying the width of n-diffusion, poly, etc).

·     Calculate the approximate resistance of interconnect based on its sheet resistance.
