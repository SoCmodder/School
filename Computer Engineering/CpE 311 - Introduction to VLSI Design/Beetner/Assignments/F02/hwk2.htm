Homework Problem 2
Due: Tue., Sept. 17.
  1. Show how to implement the carry-out function for a 1-bit adder:

    Cout = A*B + A*Cin + B*Cin

    using ACT1 logic modules (A, B, Cin, and Cout are all 1-bit variables).

  2. Implement a flip-flop using ACT1 logic modules. It does not have to have a clear or enable input.
    1. Show how the modules must be connected. Hint: Think about having the clock drive two different modules.
    2. Estimate the timing delay due only to the logic elements (i.e. the delay from a change in input to change in output) assuming a standard supply voltage and junction temperature.
    3. Estimate the complete timing delay, including delay resulting from interconnect. You can assume the logic blocks are relatively close to one another. Calculate delay as though the output of the flip-flop is unconnected (e.g. is unloaded).
    4. Estimate the maximum clock-speed. Do you believe this clock rate could be achieved in a "real" design? Why or why not?
    5. Re-estimate the timing delay if the junction temperature were 25 degrees C and the Supply votage were 5.5 volts.
  3. Briefly (4-5 sentences) explain the reason for a special-purpose clock input to an FPGA.
