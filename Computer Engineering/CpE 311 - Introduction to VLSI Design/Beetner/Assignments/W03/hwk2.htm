Homework Problem 2
Due: Tue., Feb. 11.

  1. Implement the following function using ACT1 logic modules (i.e. show connections, etc).  Try to use as few modules as possible (but make sure to get something to work).  Show your work.

F = (AÅB)+B’C

A, B, and C are 1-bit variables.  B’ is the inverse of B.

  1. Implement a flip-flop using ACT1 logic modules (i.e. show connections, etc). It does not have to have a clear or enable input.  Hint: Think about having the clock drive two different modules.
  2. Estimate the number of logic modules required to implement a registered 1-bit adder.  You do not have to have the correct number exactly.  I want you to do this to get in the practice of thinking about how much space your designs will take up in the FPGA. Hint: A registered 1-bit adder includes functions Cout = A*B + A*Cin + B*Cin (where A, B, and Cin are 1-bit inputs) and Sum = AÅBÅCin, and includes a single bit register (flip-flop) to store the resulting sum.  How many logic modules would be required to complete a 16-bit ripple-carry addition?
  3. Do the following problem from a previous test (ignore the number of points assigned to each part).  Assume Vcc=4.5V and that all horizontal tracks used in this problem are minimum length.  (You don’t have to turn this in, but what would the answer be if we used max-length horizontal tracks?)
