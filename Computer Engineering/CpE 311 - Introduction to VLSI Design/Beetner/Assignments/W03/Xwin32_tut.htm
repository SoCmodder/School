Hints on using Xwin32 to run Mentor Graphics applications

V. Shirgur, Oct. 2002 (With some edits by D. Beetner)

 

  1. Make sure you have Xwin32 installed on your machine.
  2. If a session is not configured for the computer you want to connect to, you will need to configure a session. To make a session, use the following steps:
    1. Open X Config (or X Util, as the case may be), click on Sessions, and click on Add. In the dialog box that follows, choose rexec as the Connect Method and click on Next.
    2. Give an appropriate name to the session in the next dialog box under the session field.
    3. For the host name, choose from available ECE machines, for example mhz.ece.umr.edu or euler.ece.umr.edu.
    4. Enter your login ID in the login field and the password.
    5. For the Command, copy and paste the following:

 

/usr/openwin/bin/xterm -ls -n SESSION_NAME -display $DISPLAY &

 

where SESSION_NAME is the name of your session.  DISPLAY will be filled in automatically by the host.

    1. Enter your password now if you want it saved or later if you don’t.  The other settings need not be changed.
    2. Save your session.
    3. Check that Mentor Fonts are available.  Hit the “font” tab in Xconfig.  See if the list of fonts include a font with a name like “mentor” (at the end of a long path name).  If you don’t see the mentor fonts available, copy them from the web: ftp://ftp.starnet.com/xwin32/fonts/mentorg_fonts.exe to the Xwin Font directory (typically C:\Program Files\StarNet\X-Win32\Common\Lib\ -- you should be able to see the path of other fonts in the font window) then add the mentor fonts to Xwin using the ADD button. 
    4. Click on OK to close X Config. Click on the XWin32 icon to launch XWin. Click on connection with the session name you specified and launch the XWin32 terminal.
    5. Once logged in, use typical Unix commands to work with Mentor Graphics
