The microprocessor is programmed with the game software via a serial cable from a PC COM port. The circuit board contains a serial connector and the software is transferred to the chip's memory.
Modifications or additional games could easily be written and implemented using the same hardware.

When the microprocessor receives power, it starts at a special memory location. Here it looks for the memory address indicating the beginning of the program. The microprocessor goes through essentially the same steps as a PC goes through when it is turned on. It looks for a specific memory address and loads the BIOS (stuff you see when you boot). That then tells the computer what memory address to go to to load the operating system (like Windows). Whenever you boot a PC, it always starts at the beginning and not where it left out when shut off.

Once the microprocessor has found the program, the hardware fetches the software instructions and executes them. It basically processes all of the information at the beginning of the program such as variables and constants and then begins to execute the software an instruction at a time. That's what makes everything work together. All of this is very similar to a desktop computer, without many of the complicated layers.
