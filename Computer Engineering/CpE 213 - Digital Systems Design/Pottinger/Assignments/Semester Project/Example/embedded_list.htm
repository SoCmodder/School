Common Embedded Systems

What all of these have in common, is the use of an embedded system. That is, a microcontroller based device that receives input and produces output. Although the simon game used user input, most of the embedded systems act with no user intervention. For example, an engine controller receives input from various sensors in the car, and then controls things such as engine speed, fuel consumption, etc., based upon it's input.

* microwave oven
* refrigerators
* television,VCRs, stereos
* automobiles
  -engine control, diagnostics, brakes, climate control, stereo, etc.
* Toys like virtual pets and Furby

Embedded systems are everywhere!
