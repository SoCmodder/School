Assembly language Code

When the C program is 'compiled', the C language instructions are converted into object or assembly code. The object code file is computer dependent. Each computer has it's own set of instructions that is uses to do certain things.

This game, could be played using a different computer (or microcontroller). It would first have to be 'recompiled' using a compiler specific to the new computer. For example, Windows programs have to be recompiled (and in some cases rewritten) to run on a Macintosh.

To the left, is the assembly language for ONE function, main. (it's long!) You can see all of the assembly code here. Sometimes computer programs are written entirely or partly in assembly language.


It's obvious to see the benefits of being able to write software in a high level language like C, instead of in the computer's native language.
